# Instrucciones para Subir a GitHub

## Pasos para inicializar el repositorio

1. **Inicializar Git en el directorio:**
```bash
cd servicios-module-portfolio
git init
```

2. **Agregar todos los archivos:**
```bash
git add .
```

3. **Hacer el primer commit:**
```bash
git commit -m "Initial commit: Módulo de gestión de servicios Laravel"
```

4. **Crear un nuevo repositorio en GitHub** (desde la interfaz web de GitHub)

5. **Conectar el repositorio local con GitHub:**
```bash
git remote add origin https://github.com/TU_USUARIO/TU_REPOSITORIO.git
git branch -M main
git push -u origin main
```
